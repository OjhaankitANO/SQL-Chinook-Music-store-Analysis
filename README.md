## Project Overview
Analyzed the Chinook Music Store database using SQL to derive business insights related to customer behavior, sales performance, revenue trends, and music preferences.

## Skills Used
- SQL
- Joins
- CTEs
- Window Functions
- Aggregate Functions
- Views
- Subqueries

## Key Insights
- Identified top-selling tracks and artists.
- Analyzed customer spending patterns.
- Calculated customer lifetime value.
- Examined regional sales performance.

## Database
Chinook Database
